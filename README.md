# GSM IT Solutions Website

Static website package for GSM IT Solutions.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/gsm-logo-original.png`

## Preview locally
Open `index.html` in Chrome/Edge.

## Upload to GitHub
Upload these files and folders to your repository root:
- index.html
- styles.css
- script.js
- assets/
- README.md

Do not upload this ZIP file into the repository.

## Vercel
Once the GitHub repository has these files, Vercel should deploy automatically. If not, go to Vercel > Project > Deployments > Redeploy.
