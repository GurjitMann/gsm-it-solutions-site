## Upload to GitHub
Upload these files and folders to your repository root:
- index.html
- styles.css
- script.js
- assets/
- README.md

Do not upload this ZIP file into the repository.

## Vercel
Once the GitHub repository has these files, Vercel should deploy automatically. If not, go to Vercel > Project > Deployments > Redeploy.
