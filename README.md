How to publish on Vercel:
1. Create a free Vercel account.
2. Choose Add New Project.
3. Upload this folder or connect it via GitHub.
4. Deploy.
5. In Vercel, add your custom domain: gsmitsolutions.org.
6. In Wix domain/DNS settings, point the domain to the records Vercel gives you.
